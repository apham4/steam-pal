## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Accessing the Application in Development
The application is accessible at the following URL:
```sh
http://localhost:5173/
```

### Run Unit Tests with [Vitest](https://vitest.dev/)
Remember to toggle the mock consts in api.js to true.
```sh
npm run test:unit
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
