Steam Pal is a web application designed to provide AI-powered personalized game recommendations for users on the Steam platform. 
The system leverages Google’s Gemini 2.5 Flash model to analyze a user's gaming profile and history, generating tailored suggestions. 
The application is powered by a Python FastAPI back-end, SQLite database, and a Vue/Vuetify front-end, utilizing the official Steam Web API for data integration.
